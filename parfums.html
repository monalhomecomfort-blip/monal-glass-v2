Поширені запитання

Чим парфуми для дому відрізняються від дифузорів?

Парфуми для дому діють миттєво — достатньо кількох розпилень, щоб аромат з’явився в просторі. На відміну від дифузорів, вони не діють постійно: ви самі керуєте ароматом у потрібний момент.

Скільки тримається аромат після розпилення?

Аромат одразу відчутний у повітрі, а на текстилі зберігається до 24–48 годин. Тривалість залежить від розміру приміщення, вентиляції та кількості розпилень.

Чи можна регулювати інтенсивність аромату?

Так. 1–2 розпилення — легкий фон. Більше — більш насичений аромат без перевантаження простору.

Для яких приміщень підходять парфуми для дому?

Підійдуть для вітальні, спальні, коридору, гардеробної або робочого простору — всюди, де хочеться швидко змінити атмосферу.

Чи безпечні парфуми для дому?

Парфуми для дому Mōnal призначені для використання в житлових приміщеннях. Рекомендуємо не розпилювати безпосередньо на людей, тварин та меблі.

Чи підійдуть парфуми для дому на подарунок?

Так. Парфуми для дому — універсальний подарунок, який легко використовувати і приємно дарувати.
